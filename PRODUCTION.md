## Перед деплоем

1. **Обновите домен в файлах:**
   - `src/robots.txt` - замените `yourdomain.com` на реальный домен
   - `src/sitemap.xml` - замените `yourdomain.com` на реальный домен
   - `src/components/layout/head/head.html` - обновите og:url и og:image

2. **Проверьте формы:**
   - Настройте отправку форм (PHP/Telegram/Email)
   - Проверьте валидацию всех полей

3. **Сборка для продакшена:**
   ```bash
   npm run build
   ```
   Результат будет в папке `dist/`

4. **Проверьте:**
   - Все ссылки работают
   - Формы отправляются
   - Изображения загружаются
   - Модалки открываются
   - Навигация работает на всех устройствах
